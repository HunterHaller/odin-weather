# odin-weather
Weather app project as assigned in The Odin Project

## Features
- Shows the weather in a style similar to the old Wii Forecast Channel!
- Fetches weather data from VisualCrossing.
- Dynamically loads weather icons based on results.
- Waits for icon images to load before displaying them.
- Animates icon entrance
- Shows loading text while waiting for API call to resolve.
- Allows users to search for weather data from their location of choice.
